# port-terminator

Because we need a terminator to kill programs on our ports!

## Install

Clone, copy, download the shell script. Run in your terminal.

## running

You can pass a port as an argument:

`./terminate.sh 8000` 

Or `./terminate.sh` to prompt you for a port.
